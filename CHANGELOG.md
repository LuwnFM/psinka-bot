# История версий

## [0.3.0] — 2026
### 🎉 Добавлено
- ✅ Полная миграция на базу данных Neon Tech (PostgreSQL)
- ✅ Слэш-команда `/кубик` с поддержкой синтаксиса DiceMaiden
- ✅ Команда `/статус` для просмотра статистики бота и БД
- ✅ Проверка доступа по Discord роли (переменная `ROLE_ID`)
- ✅ Продвинутый парсер кубиков: взрывы (e/ie), удержание (k/kl), сброс (d), перебросы (r/ir), успехи (t/f/b)
- ✅ Приватные броски (`p`), упрощённый вывод (`s`, `nr`), комментарии (`!`)

### 🔧 Изменено
- 🔄 Удалена зависимость от `test_history.json` — вся история теперь в БД
- 🔄 Команда `кубик` переведена на слэш-команды (DiceMaiden стиль)
- 🔄 Команда `лучшие` удалена (заменена на `/статус`)
- 🔄 Обновлена структура хранения успешных запросов к моделям
- 🔄 Улучшена обработка ошибок и логирование

### ❌ Удалено
- 🗑 Файл `test_history.json` (больше не используется)
- 🗑 Команда `псинка лучшие`
- 🗑 Команда `псинка скачать_историю`
- 🗑 Команда `очистить` (purge) — нет прав у бота

### ⚙️ Технические изменения
- 📦 Добавлены зависимости: `psycopg2-binary`, `sqlalchemy`
- 🗄 Создана таблица `model_success_log` в Neon DB
- 🔐 Добавлена проверка роли через переменную `ROLE_ID`
- 📊 Автоматическая очистка старых записей в БД (>200 записей)
