# 🐕 PsIInka Bot — Changelog

Все значимые изменения в проекте будут задокументированы в этом файле.

---

## [0.2.0] — 2026

### 🆕 Новое
- ✅ **Поддержка прокси** — интеграция с ProxyScrape Official API для обхода блокировок
- ✅ **Автоматическое обновление прокси** — список обновляется при старте бота
- ✅ **Настройка через переменные** — включение/выключение через `USE_PROXY` в Railway
- ✅ **Безопасные запросы** — прокси используются только через официальные API (без нарушения ToS)

### ⚙️ Оптимизация
- 🔧 Улучшена обработка таймаутов при запросах к OpenRouter
- 🔧 Добавлена защита от переполнения файлов логов
- 🔧 Уменьшено потребление памяти при работе с `test_history.json`

### 🐛 Исправления
- 🐞 Исправлена ошибка записи файлов на Railway (временная файловая система)
- 🐞 Улучшена стабильность команды `скажи` при неудачных запросах
- 🐞 Добавлены подробные логи для отладки прокси-соединений

### 📋 Переменные окружения
| Ключ | Значение по умолчанию | Описание |
|------|----------------------|----------|
| `USE_PROXY` | `false` | Включить обход блокировок через прокси |
| `RAILWAY` | `true` | Режим хостинга Railway |
| `MAX_CONCURRENT` | `2` | Лимит параллельных запросов |

## [0.1.0] — 2026

### 🆕 Новое
- ✅ Первый релиз бота
- ✅ Команды: `погавкай`, `кубик`, `тест`, `скажи`, `лучшие`
- ✅ Интеграция с g4f и OpenRouter API
- ✅ Тестирование моделей с прогресс-баром
- ✅ Сохранение истории тестов в `test_history.json`
- ✅ Оптимизация для Railway (без полного теста)

### 📋 Требования
- Python 3.9+
- disnake, g4f, openai, aiohttp, python-dotenv
