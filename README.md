  🐾 О ПРОЕКТЕ 🐾

  ════════════════════════════════════════════════════════════════════════════

  Это экспериментальный проект в стиле vibecoding — кодирование по наитию,
  без строгих правил и методологий. Мы пробуем, тестируем, ошибаемся и
  находим работающие решения через эксперименты.

  Бот создан для тестирования AI-моделей через g4f и OpenRouter API с
  возможностью обхода блокировок и автоматического выбора лучших провайдеров.

  🦴 VIBECODING - ЭТО... 🦴

  ════════════════════════════════════════════════════════════════════════════

  • Кодим когда хочется и чё по кайфу, а не когда чёт надо

  • Ломаем, чиним, снова ломаем
  • Находим работающие решения через эксперименты

  Версии могут быть нестабильными. Используйте на свой страх и риск.

  🦴 УСТАНОВКА 🦴

  ════════════════════════════════════════════════════════════════════════════

  1. Клонируйте репозиторий:

     git clone https://github.com/LuwnFM/psinka-bot.git
     cd psinka-bot

  2. Установите зависимости:

     pip install -r requirements.txt

  3. Создайте файл .env:

     DISCORD_TOKEN=ваш_токен_бота
     OPENR_TOKEN=ваш_токен_openrouter
     OWNER_ID=ваш_discord_id
     USE_PROXY=false
     MAX_CONCURRENT=2

  4. Запустите бота:

     python psinkamain.py


  ════════════════════════════════════════════════════════════════════════════

  🐾 НАСТРОЙКА RAILWAY 🐾

  ════════════════════════════════════════════════════════════════════════════

  1. Создайте проект на railway.app
  2. Выберите Deploy from GitHub repo
  3. Выберите репозиторий LuwnFM/psinka-bot
  4. Добавьте переменные окружения:

  ┌──────────────────────────────────────────────────────────────────────────┐
  │  Переменная      │  Значение           │  Описание                      │
  ├──────────────────────────────────────────────────────────────────────────┤
  │  DISCORD_TOKEN   │  ваш_токен          │  Токен бота из Discord Portal  │
  │  OPENR_TOKEN     │  ваш_токен          │  Токен от OpenRouter API       │
  │  OWNER_ID        │  ваш_ID             │  Discord ID владельца          │
  │  USE_PROXY       │  false              │  Включить прокси (опционально) │
  │  MAX_CONCURRENT  │  2                  │  Лимит параллельных запросов   │
  │  RAILWAY         │  true               │  Режим хостинга Railway        │
  └──────────────────────────────────────────────────────────────────────────┘

  5. В Settings укажите Start Command: python psinkamain.py
  6. Сохраните и дождитесь запуска
